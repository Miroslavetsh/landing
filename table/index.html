Дуже простий та зручний

Виконує широкий спектр функцій, таких як підставка для ноутбука, стіл для сніданку, столик в ліжко, письмовий стіл.Столик портативний та досить легкий, що дозволяє переставляти його на інше місце та робить ідеальним для подорожей.

Оснащений ніжками з захистом від ковзаня, які зафіксують положення столу практично на будь-якій поверхні. Не займає багато місця, легко і компактно складається до товщини 4 см, його легко заховати у шафі, за шафою, під ліжком...


Портативний: підходить для студентів, офісних працівників, дітей, читання, мандрівок і т.д.


Переваги

  • Ніжки які не ковзають та забезпечують надійну фіксацію на будь-якій поверхні.

  • Підставка для стакану дозволяє мати напій завжди під рукою для більш комфортної роботи чи відпочинку.

  • Легкі металеві ніжки складаються, що робить столик компактним та зручним для зберігання та перенесення.
